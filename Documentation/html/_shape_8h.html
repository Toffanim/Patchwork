Detailed Description

Header files containing all things related to mathematics (low level).

Header files containing all things related to geometric shapes.

Provides trogonometric functions : fast_sin, fast_cos, fast_sqrt Provides a 2D vector classe (also used for containing points), and vector algebra. Provides a Color container.

Gives access to the classes : Circle, Polygon, Line and Ellipse, which all inherits from the Shape class. Also provides the Image class which is a container of shapes.
